# PINBRIDGE Persistence Smoke Checks

Run in a browser with devtools open.

1. Create a note with a title and body.
2. Reload the page.
3. Confirm the note still exists with the same content.
4. Edit the note content.
5. Reload the page.
6. Confirm edits persist.
7. Log out, log back in, and confirm notes load.

Expected: No note loss or silent reset between reloads, unlocks, or logouts.
